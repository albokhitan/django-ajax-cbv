## How to install
To install the app run :
```shell
pip install django-ajax-views
```
or add it to the list of requirements of your project.

Then add 'ajax\_views' to your INSTALLED_APPS.
```python
INSTALLED_APPS = [
    ...
    'ajax_views',
]
```
