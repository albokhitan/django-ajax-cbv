Django module to easily use ajax forms.

## Install
To install the app run :
```shell
pip install django-ajax-views
```
or add it to the list of requirements of your project.

## Running tests
To run tests we use tox:
```
# Run unit and flake
$ tox

# Run only unit tests
$ tox -e py27

# Run only flake8 tests
$ tox -e flake8

# Run coverage tests
$ tox -e coverage
```
